# protogo

**protogo** is a career, gig, and business focused application (front-end) for the Fediverse and Social Web.

## Building

To build, you may need to do:

```bash
sudo apt update
sudo apt install libxkbcommon-x11-dev libx11-xcb-dev libwayland-dev libegl1-mesa-dev libxcursor-dev libxfixes-dev
```

Or:

```
sudo dnf install libxkbcommon-x11-devel libX11-xcb-devel wayland-devel mesa-libEGL-devel libXcursor-devel libXfixes-devel
```
